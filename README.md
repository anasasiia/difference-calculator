Вычислитель отличий - это программа, которая сравнивает 2 структуры данных (формат .json или .yaml) и выводит в консоль отличия в 3 форматах на выбор: stylish, plain или json.

### Setup

```
make build
```

### Run

```
make run
```

### Run tests

```
make test
```

### Run checkstyle

```
make lint
```
